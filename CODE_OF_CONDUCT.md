# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, as its code of conduct.
The full text is available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

By participating in this project — as a contributor, maintainer, issue reporter, or
reviewer — you agree to uphold this code in all project spaces (issues, pull requests,
discussions, mailing lists, conference interactions, and any other communication
channel related to the project).

## Reporting

Concerns or violations may be reported confidentially to the project maintainer at
<koffigodwin96@gmail.com>. Reports are reviewed and addressed promptly. The maintainer
is obligated to respect the privacy and security of the reporter.

## Enforcement

Maintainers are responsible for clarifying and enforcing standards of acceptable
behaviour. They may take corrective action in response to any conduct deemed
inappropriate, threatening, offensive, or harmful, following the
[enforcement guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines)
of the Contributor Covenant 2.1.

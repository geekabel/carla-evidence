# RFCs (Request for Comments)

Design documents for non-trivial library decisions. Each RFC captures the
context, the candidate options, the chosen design, and the trade-offs we
accepted on the way. Once an RFC ships, it becomes a frozen historical record;
later changes go into a new RFC that references the original.
